1- Description générale

<<<<<<< .mine

Nous nous sommes placés dans la position d’un regroupement de personnes passionnées de lapins. La création d’un site pour aider les propriétaires de lapins à les faire reproduire n’existe pas encore et pourrait être utile par exemple pour la sauvegarde de certaine race. L’objectif de ce projet est donc de créer un site de rencontre pour les lapins à la manière des réseaux sociaux afin de favoriser les échanges et rencontres entre lapins par le biais d’une inscription sur le site, d’outils génériques de discussion, organisation d’événements (concours…). Le concept de ce site serait de réaliser un site interactif avec une partie modérateur. Les propriétaires des lapins pourront correspondre avec d'autres propriétaires de lapins. Notre site n’est pas à but lucratif pour le moment. Il pourrait le devenir par la suite si l’on décide de créer des partenariats avec des marques commerciales par la suite (équipement de lapins, nourriture pour lapins, partenariats avec des concours etc.)

Objectifs secondaires :

  • Développer le nombre de clients
  • Fidéliser ses clients
  • Rendre ce site de rencontre visible sur internet
  • Déployer l’observation sociale en Région Centre
